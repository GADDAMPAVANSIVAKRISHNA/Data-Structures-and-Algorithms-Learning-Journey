# Python Basic Syntax

## Concept

Python uses simple syntax and automatic typing, which makes code easy to read.

- You assign values directly without declaring types.
- Indentation defines code blocks instead of braces.
- Use `print()` for output and `input()` for input.

## Examples

### 1. Variables and values

```python
x = 10
message = "Hello"
pi = 3.14
print(x, message, pi)
```

### 2. Conditional statement

```python
if x > 5:
    print("x is greater than 5")
else:
    print("x is 5 or less")
```

### 3. Loop with range

```python
for i in range(3):
    print(i)
```

## Practice Questions

1. Create two variables and print their sum.
2. Write an `if` statement that prints "even" when a number is divisible by 2.
3. Use `range(1, 6)` to print numbers from 1 to 5.
