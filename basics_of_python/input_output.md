# Input / Output in Python

## Concept

Input and output let your program communicate with the user.

- `input()` reads text typed by the user.
- `print()` shows text or values on the screen.
- Get data from the user, then display results clearly.

## Examples

### 1. Read text and show it

```python
name = input("Enter your name: ")
print("Hello, " + name + "!")
```

### 2. Read numbers and add them

```python
x = int(input("Enter first number: "))
y = int(input("Enter second number: "))
print("Sum:", x + y)
```

### 3. Print using f-strings

```python
age = 25
print(f"You are {age} years old.")
```
